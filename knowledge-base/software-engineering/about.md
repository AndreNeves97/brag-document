# Software Engineering

General notes on software engineering principles, practices, and methodologies.

## Internal Resources

- **[Hacker Laws](hacker-laws.md)**: A comprehensive list of laws, theories, principles, and patterns that developers should know (e.g., Murphy's Law, Brooks' Law, SOLID).

## External Resources

- **[Refactoring.guru](https://refactoring.guru/)**: Excellent resource for Design Patterns and Refactoring techniques.
- **[Martin Fowler](https://martinfowler.com/)**: Articles on software architecture, microservices, and agile development.
- **[The Twelve-Factor App](https://12factor.net/)**: A methodology for building software-as-a-service apps.
- **[Clean Code Blog](https://blog.cleancoder.com/)**: Uncle Bob's blog on software professionalism and craftsmanship.
